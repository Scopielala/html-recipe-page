# 🍛 Nigerian Jollof Rice Recipe Page

This project is a simple HTML-based recipe that showcases how to structure content using semantic HTML elements. The page presents a detailed guide on how to prepare Nigerian Jollof Rice, including ingredients, cooking instructions, and nutritional information.

---

## 📌 Project Overview

The goal of this project is to practise:

- Writing clean and structured HTML
- Using semantic tags appropriately
- Organizing content into meaningful sections
- Displaying tabular data using HTML tables
- Embedding images and links

---

## 🧱 Built With

- HTML (No CSS or Javascript used)

---

## 📄 Features

- 📖 Detailed description of Nigerian Jollof Rice
- 📝 Clearly structured recipe sections:
  - Recipe Details
  - Ingredients
  - Instructions
  - Tips & Notes
  - Nutrition Facts
  - Recipe Source
- 📊 Tables for recipe details and nutritional values
- 🖼️ Image included with caption
- 🔗 External link to original recipe source

---

## How to view

1. Clone the repository:
   git clone https://github.com/Scopielala/html-recipe-page.git
2. Open the project folder
3. Double-click on index.html or open it in your browser

## Learning Objectives

This project demonstrates:

- Understanding of semantic HTML elements like <main> <article>, <section>, <table>, <ul>, and <ol>
- Structuring content for readability and accessibility
- Basic use of HTML attributes such as alt, href, and meta viewport
